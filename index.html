Core Features

Everything You Need in One Place

Nightingale v2.0 brings a revolutionary web-based interface with powerful features designed for modern penetration testing workflows.

Multi-Terminal Support

Manage multiple terminal sessions simultaneously with real-time synchronization. Switch between sessions seamlessly and execute commands across different contexts.

Web-Based GUI

Access your entire penetration testing environment through a modern, intuitive web interface. No local installations required - everything runs in your browser.

AI Command Suggestions

Get intelligent command recommendations based on your current context and output. AI-powered suggestions help you discover new tools and techniques.

Real-Time File Explorer

Navigate your file system with an interactive, expandable directory tree. Real-time updates keep you in sync with your terminal's current working directory.

VPN Management

Built-in VPN support for connecting to testing networks. Manage VPN connections directly from the interface with split-tunneling capabilities.

Scan Management

Organize and manage your vulnerability scans with a comprehensive dashboard. Track scan progress, view results, and manage scan history efficiently.

Fast Boot Time

Get started in seconds with Docker's lightweight virtualization. No need to install multiple programming languages or configure complex environments.

Platform Independent

Run on any platform that supports Docker - Windows, macOS, Linux, or cloud. Your environment stays consistent regardless of the host operating system.
